# Server HTTP 

## Verbs (Request)

- GET -> request data
- POST -> register new data
- PUT/PATCH -> update data
- DELETE -> delete data

## Status

- 200 -> Sucess for GET
- 201 -> Sucess for POST

- 400 -> Invalid syntax
- 401 -> Necessary autentication
- 403 -> Access denied
- 404 -> Information not found on the server
- 422 -> Semantic errors
